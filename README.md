# 🌍 AI Travel Planner

An AI-powered travel planning app built using CrewAI agents.

## 🚀 Features
- Multi-agent system (Researcher, Budget Planner, Itinerary Planner)
- Generates travel plans based on budget & interests
- 3-day itinerary generation
- PDF export

## 🛠 Tech Stack
- Python
- Streamlit
- CrewAI
- Gemini API

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
